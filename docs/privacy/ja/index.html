プライバシーポリシー

最終更新日: 2026年6月16日

このプライバシーポリシーは、The Kanji の利用時に情報をどのように扱うかを説明します。 The Kanji は日本語単語クイズアプリです。クイズの進捗とアプリ設定を端末内に保存し、 アプリの利用状況を把握するために Firebase Analytics を使用します。

端末内に保存される情報

アプリは、機能を提供するために以下の情報を端末内に保存します。

  • オンボーディング完了状態
  • 言語設定
  • 文字サイズ設定
  • サウンド設定
  • デイリークイズ完了状態
  • クイズスコア、連続記録、総セッション数、および関連する進捗統計

これらの端末内情報は、進捗、連続記録、設定、デイリー完了状態などを表示するために使用されます。 現在のアプリは、アカウントログイン、クラウド同期、サーバー側の進捗復元を提供していません。

Analytics

The Kanji は Firebase Analytics を使用して、アプリの利用状況に関する analytics data を収集します。 これは、アプリの使われ方を把握し、品質と体験を改善するために使用されます。

analytics event には、クイズ開始、クイズ完了、機能利用、画面利用、アプリバージョン、 端末モデル、OSバージョン、言語または地域設定、おおよその利用タイミングなどが含まれる場合があります。

現在の analytics 実装は、ユーザーID、アカウント情報、広告識別子、問題本文、回答本文、 支払い情報、連絡先情報、セッションリプレイ録画を意図的に送信しません。

Firebase Analytics は Google Firebase によって提供されています。Google は Firebase および Google の規約とプライバシー慣行に従って analytics data を処理する場合があります。

収集しない情報

現在のアプリは、以下を収集またはアップロードしません。

  • アカウント登録情報
  • ログイン認証情報、氏名、メールアドレス、プロフィール情報
  • 支払いまたは購入情報
  • ユーザー生成コンテンツ
  • クラウド同期されるクイズ進捗

デバイス権限

現在のアプリは、カメラ、マイク、位置情報、連絡先、写真ライブラリ、App Tracking 権限を要求しません。

広告

The Kanji は Google AdMob および Google Mobile Ads SDK を使用して広告を表示する場合があります。 広告SDKは、広告の配信、測定、不正防止、広告品質の改善のために、IPアドレスから推定されるおおよその位置情報、 端末識別子または広告識別子、広告の表示・クリック・動画視聴などの広告利用データ、 アプリ起動や広告表示に関連する利用情報、クラッシュログ、パフォーマンスデータを収集または処理する場合があります。

これらのデータは Google および広告配信に関わる事業者によって処理される場合があります。 パーソナライズ広告や地域ごとの同意が必要な場合、アプリ内またはOSの設定で広告・トラッキングに関する選択肢が表示されることがあります。 現在のアプリは、広告SDKに問題本文、回答本文、アカウント情報、支払い情報、連絡先情報を意図的に送信しません。

購入

現在のアプリには、アプリ内課金、サブスクリプション、ペイウォール、購入復元は含まれていません。

子ども向け対応

The Kanji は一般向けの日本語単語クイズアプリです。アカウント作成、ソーシャル機能、 ダイレクトメッセージ、ユーザー生成コンテンツは含まれていません。

ポリシーの変更

analytics、購入、広告、クラッシュレポート、アカウント機能、クラウド同期、 その他ネットワークサービスなど、アプリの変更に応じてこのプライバシーポリシーを更新する場合があります。 更新後のポリシーはこのURLで公開されます。
